# Financial Activity Dashboard

A responsive frontend dashboard built with React to explore, filter, and visualize financial transaction data.

## Overview

The Financial Activity Dashboard focuses on presenting transactional data in a clear and intuitive user interface.  
It allows users to view transactions, apply filters, and understand spending patterns through summaries and visualizations.

The goal of this project is to demonstrate frontend fundamentals such as component design, state management, data handling, and UI usability.

## Features

- Display financial transactions in a structured table
- Filter and sort transactions by date, category, and type
- Summary view of income, expenses, and balance
- Visual representation of transaction data using charts
- Responsive layout for desktop and tablet screens
- Loading, empty, and error states for better user experience

## Tech Stack

- React
- JavaScript (ES6+)
- HTML & CSS
- Vite (development and build tool)
- Mock REST API (json-server)
- Chart library (for data visualization)

## Data Source

Transaction data is served through a mock REST API using `json-server` to simulate real backend interactions.
